# Adding Screenshots

Here is how to embed screenshots in this documentation:

1) Create the folder: `docs/images/`
2) Place images there, e.g., `dashboard.png`, `features-list.png`, …
3) Reference them in the Markdown files, e.g.:
   - `![Dashboard](images/dashboard.png)`

Recommendations
- Consistent resolution (e.g., 1600×900)
- Use anonymized data
- Descriptive file names (e.g., `vote-session-cards.png`)

Placeholders already set in: `user-guide.md`
