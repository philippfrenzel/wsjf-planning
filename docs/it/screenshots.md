# Aggiungere screenshot

Per integrare screenshot in questa documentazione:

1) Creare la cartella: `docs/images/`
2) Inserire le immagini, ad es. `dashboard.png`, `features-list.png`, …
3) Referenziare nei file Markdown, ad es.:
   - `![Dashboard](images/dashboard.png)`

Raccomandazioni
- Risoluzione uniforme (ad es. 1600×900)
- Utilizzare dati anonimizzati
- Nomi di file significativi (ad es. `vote-session-cards.png`)

Segnaposto già impostati in: `benutzerhandbuch.md`
