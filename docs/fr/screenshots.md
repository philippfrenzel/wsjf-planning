# Ajouter des captures d'écran

Voici comment intégrer des captures d'écran dans cette documentation :

1) Créer le dossier : `docs/images/`
2) Déposer les images, par ex. `dashboard.png`, `features-list.png`, …
3) Les référencer dans les fichiers Markdown, par ex. :
   - `![Dashboard](images/dashboard.png)`

Recommandations
- Résolution uniforme (par ex. 1600×900)
- Utiliser des données anonymisées
- Noms de fichiers descriptifs (par ex. `vote-session-cards.png`)
